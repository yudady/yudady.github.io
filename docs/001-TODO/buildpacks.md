## 是什麼

- Pack 是 Cloud Native Buildpacks 項目維護的一個工具，用於支持使用 buildpacks。
- 不用寫 Dockfile ， pack 會偵測到 maven ,打包成 OCI images

## 去哪下載

- choco install pack --version=0.27.0
