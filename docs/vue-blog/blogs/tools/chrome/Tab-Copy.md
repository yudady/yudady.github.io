## TabCopy使用簡介

- 有兩個模式：Simple Mode/Fancy Mode
- 在Simple Mode選用Markdown格式
- [title] 會擷取網頁標題
- [url]擷取網
- Fancy Mode可設定3個格式，將其中一個換成Custom，Tab內容設定如下：



```code
`- [[title]] [title]([url])`
```
