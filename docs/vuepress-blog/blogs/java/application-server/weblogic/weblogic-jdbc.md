# 簡介

創建weblogic.datasource

> JDBC 資料來源是一項與 JNDI 樹狀結構相連結的物件，JNDI 樹狀結構透過 JDBC 連線的集區提供資料庫連線能力。應用程式可以在 JNDI 樹狀結構中尋找資料來源，然後借用資料來源的資料庫連線。
